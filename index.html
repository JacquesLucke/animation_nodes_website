Get Started

  • Read the Animation Nodes Getting Started Guide.
  • Check out the community-made tutorials from the list of tutorial creators.
  • Get help by asking questions on StackExchange using the animation-nodes tag. Read the 500+ answers already available!
  • Get support by filling issues on GitHub. Report bugs, submit feature requests, and discuss Animation Nodes with the developers.
  • See what is new by reading the Release Notes and the Change Log.
  • Get the source code from Github and get involved by reading the Developers Guide.
